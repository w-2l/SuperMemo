Incremental reading: Importing articles

The first step in incremental learning is to import an article to read and learn. Usually you will import an article directly from the Internet.

There are 2 basic ways of importing an article:

  1. One article - copy the selected text in your web browser to the clipboard, and use (Ctrl+N) in SuperMemo
  2. Many articles - open many articles in the web browser, and use Edit : Web import (Shift+F8) in SuperMemo. You will be able to import the selected articles.

    Hint: If you cannot see Edit : Web import in the main menu, you need to switch SuperMemo to File : Level : Middle or higher level for the option to become available.
