Try to import an article now!

For example:

  1. Open any article in your web browser (for example, right click over this link https://help.supermemo.org/wiki/Incremental_reading and choose Open in new window)
  2. Select any portion of text with the mouse (in the browser)
  3. Press Ctrl+C (in the browser)
  4. Return to SuperMemo (e.g. with Alt+Tab)
  5. Press Ctrl+N (in SuperMemo)

Hint: After you have imported an article, you can always return to this page by using the button (Alt+Left arrow).
